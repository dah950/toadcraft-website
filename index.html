Server Rulz

  • No Griefing: Do not destroy or modify other players' builds or property without their explicit permission.
  • Be Nice: Treat all players with respect. Harassment, discrimination, or any form of hateful speech will not be tolerated.

How to Join

Java Edition (PC)

  1. Open your Minecraft Java Edition game.
  2. Click on "Multiplayer".
  3. Click on "Add Server".
  4. For the Vanilla+ Server, enter the Server Address:
    mc.toadhead.net
  5. For the Evolution Server, enter the Server Address:
    mc.toadhead.net:25595 (subject to change to evo.mc.toadhead.net)
  6. Click "Done" or "Join Server".
  7. Scroll down to see how to exit the Lobby and join actual servers.

Bedrock Edition (Mobile & Windows versions)

Our Vanilla+ server supports Bedrock Edition players!

Note: The Evolution server does not currently support Bedrock Edition.


  1. Open Minecraft Bedrock Edition on your device.
  2. Go to the "Play" tab.
  3. Navigate to the "Servers" tab.
  4. Scroll down and click on "Add Server".
  5. Enter the following details:
    • Server Name: Any server name you want (e.g., "Toadhead")
    • Server Address: mc.toadhead.net
    • Port: 19132 (This is the default Bedrock port, usually automatically filled)
  6. Click "Save" or "Play".
  7. You should now see the server in your list and can connect!
  8. Scroll down to see how to exit the Lobby and join actual servers.

Bedrock Edition (Xbox, PlayStation, Nintendo Switch)

Note: You must have whatever online subscription your device has. Ie. Nintendo Online for Switch, Xbox Game Pass for Xbox, etc.

Connecting to custom servers on consoles requires a small workaround, as consoles do not allow direct server IP input. Here is a common method:

DNS Trick (Recommended)

This method involves changing your console's DNS settings to redirect to a public server list, which then allows you to connect to custom IPs.

  1. On your console, go to your **Network Settings**.
  2. Find **DNS Settings** (sometimes under Advanced Settings or Manual Setup).
  3. Change your **Primary DNS** to: 104.238.130.180 (This is a common public DNS for BedrockConnect. **Please replace with a known working DNS if you have one and this one does not work.**)
  4. Set your **Secondary DNS** to 8.8.8.8 (Google's public DNS) or leave it blank if not required.
  5. Save the settings and test your internet connection.
  6. Open Minecraft Bedrock Edition. Go to the "Servers" tab.
  7. You should now see a list of featured servers. Click on any one. If you do join the actual server and don't see a prompt to enter a server ip, try a diferent one. If all of them don't work, use a different DNS
  8. Once connected to the custom server list, you can enter our Vanilla+ server address: (mc.toadhead.net) and the port (19132).
  9. Scroll down to see how to exit the Lobby and join actual servers.

Important: You may need to revert your DNS settings to automatic after playing on custom servers if it causes issues with other online games or services.
